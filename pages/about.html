About DisFrame

A framework for building better Discord bots

What is DisFrame?

DisFrame is a flexible, extensible Discord bot built with Python and discord.py. Designed by SirCryptic, it empowers users with a modular framework for creating custom bot experiences.

Check out our community cogs repository to explore or contribute additional features!

Modular Design

Easily add commands via Python cogs in the cmds/ directory.

Role-Based Access

Permissions for owners, devs, mods, and subscribers.

Moderation Tools

Kick, ban, mute, warn, automod, and event logging.

User Engagement

Memes, translations, profiles, and reaction roles.

Multi-Guild Support

Configure features across multiple servers.

Games

Fun interactive games to engage your community.
