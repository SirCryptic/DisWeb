Hosting Guide

Run your own DisFrame bot instance

Get Started with DisFrame

Follow these steps to host your own Python-based Discord bot.

1. Prerequisites

Install Python 3.8+ and Git.

2. Download

Download DisFrame from GitHub.

3. Setup

Unzip, open a terminal in the folder, and run:
pip install -r requirements.txt

4. Configure

Create config.json with your bot token from the Discord Developer Portal:
{"token": "YOUR_BOT_TOKEN"}

5. Launch

Run the bot:
python main.py

6. Customize

Add cogs to cmds/. Use load, unload, or reload in Discord.

7. Host 24/7

Use a VPS or Replit for persistent hosting.
