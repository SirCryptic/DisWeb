About DisFrame

A powerful framework for crafting custom Discord bots

What is DisFrame?

DisFrame is an extensible Discord bot framework built with Python and discord.py, created by SirCryptic. It offers a modular architecture that empowers users to tailor bot functionality to their needs, from advanced moderation to interactive community features.

Explore and contribute to our community cogs repository to enhance DisFrame with your own creations!

Modular Design

Add or modify commands effortlessly with Python cogs in the cmds/ directory.

Role-Based Access

Granular permissions for owners, devs, mods, subscribers, and beyond.

Moderation Tools

Robust features like kick, ban, mute, warn, auto-moderation, and guild setup.

User Engagement

Interactive tools including memes, translations, profiles, and reaction roles.

Multi-Guild Support

Customize settings per guild, such as NSFW content or moderation rules.

Persistent Settings

Bot lock and DM states preserved across restarts using JSON storage.
